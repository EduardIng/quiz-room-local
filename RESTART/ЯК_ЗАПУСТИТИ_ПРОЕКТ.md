# 🚀 Як запустити проект Quiz Room Local

> Читай уважно, роби крок за кроком. Не пропускай кроки!

---

## Що таке термінал?

Термінал — це чорне (або біле) вікно де ти пишеш команди і комп'ютер їх виконує.
На Mac його можна відкрити так: натисни **CMD + Пробіл**, напиши **Terminal**, натисни **Enter**.

---

## КРОК 1 — Відкрий перший термінал (для бекенду)

1. Відкрий термінал
2. Напиши цю команду і натисни **Enter**:
```
cd ~/quiz-room-local
```
> Це означає "перейди в папку проекту"

3. Тепер напиши цю команду і натисни **Enter**:
```
npm start
```
> Це запускає сервер (мозок гри). Ти побачиш текст що з'явиться — це нормально, сервер працює.

⚠️ **Важливо:** НЕ закривай це вікно! Воно має працювати весь час поки ти розробляєш.

---

## КРОК 2 — Відкрий другий термінал (для фронтенду)

1. Відкрий **НОВИЙ** термінал (перший залишається відкритим!)
   - На Mac: натисни **CMD + T** щоб відкрити нову вкладку в тому ж терміналі
2. Напиши цю команду і натисни **Enter**:
```
cd ~/quiz-room-local/frontend
```
3. Тепер напиши цю команду і натисни **Enter**:
```
npm run dev
```
> Це запускає інтерфейс сайту. Ти побачиш посилання типу `http://localhost:5173` — це твій сайт.

⚠️ **Важливо:** НЕ закривай і це вікно!

---

## КРОК 3 — Відкрий третій термінал (для Claude Code — твого AI помічника)

1. Відкрий **ЩЕ ОДИН** новий термінал (CMD + T)
2. Напиши цю команду і натисни **Enter**:
```
cd ~/quiz-room-local
```
3. Тепер напиши цю команду і натисни **Enter**:
```
claude --dangerously-skip-permissions
```
> Це відкриє Claude Code — твій AI помічник який знає проект і допомагає з кодом.

4. Claude автоматично прочитає файл `CLAUDE.md` і знатиме правила проекту.
5. Потім скажи йому (скопіюй і встав це повідомлення — просто змінити останній рядок):

```
Прочитай CLAUDE.md і PROGRESS.md і продовжуй розробку.
Ось що я хочу зробити далі: [напиши тут своє завдання]
```

> Файл PROGRESS.md містить повний журнал всього що було зроблено — Claude прочитає його і буде точно знати де ми зупинились.

---

## КРОК 4 — Перевір що все працює

Відкрий браузер і перейди на:
- **Планшет гравця:** http://localhost:5173/#/
- **Хост (ведучий):** http://localhost:5173/#/host
- **Великий екран:** http://localhost:5173/#/screen
- **Створення квізу:** http://localhost:5173/#/create
- **Статистика:** http://localhost:5173/#/stats

Якщо сторінки відкриваються — все працює! 🎉

## Якщо щось пішло не так

**Помилка "command not found: node"**
→ Node.js не встановлений. Напиши в Claude: "встанови node"

**Помилка "command not found: claude"**
→ Claude Code не встановлений. Іди на https://claude.ai/code і встанови його.

**Сторінка не відкривається в браузері**
→ Перевір чи обидва перші термінали запущені і не показують помилок.

**Сервер вже запущений (помилка "port in use")**
→ Напиши в терміналі:
```
lsof -ti:8080 | xargs kill -9
```
і запусти знову.

**Фронтенд вже запущений (помилка "port in use" на 5173)**
→ Напиши в терміналі:
```
lsof -ti:5173 | xargs kill -9
```
і запусти знову.
